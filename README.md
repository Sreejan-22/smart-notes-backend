# smart-notes-backend

## Auth Routes

| Route   | Request Method | description                 |
| ------- | -------------- | --------------------------- |
| /signup | GET            | sign up page                |
| /login  | GET            | log in page                 |
| /singup | POST           | create a new user in db     |
| /login  | POST           | authenticate a current user |
| /logout | GET            | log a user out              |
